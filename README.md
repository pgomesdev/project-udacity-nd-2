This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

To run this project follow these steps:

1. **Clone server repo**

Go to https://github.com/udacity/reactnd-project-readable-starter, clone the repo and follow the instructions to run a local server

2. **Clone this repo**

Clone this repo to your machine

3. **Install dependencies**

Now install the dependencies of the project.

NPM:
`npm i`

Yarn:
`yarn`

4. **Run the project**

Now it's all good to run the project.

NPM:
`npm start`

YARN:
`yarn start`
